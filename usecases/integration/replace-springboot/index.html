With its tailored data access layer for relational databases, Ballerina provides a seamless developer experience, making it the natural choice for REST API development with relational data stores.

OpenAPI; from spec to code and backwards

Ballerina OpenAPI tool supports generating client/service code using OpenAPI specification. It also can generate an OpenAPI specification for a given REST service.

Learn more about Ballerina OpenAPI tool
// Generate service code using Ballerina OpenAPI tool
bal openapi -i social_media_service.yaml --mode service

// Generate client code using Ballerina OpenAPI tool
bal openapi -i social_media_service.yaml --mode client

// Generate OpenAPI specificatoin using service code
bal openapi -i social_media_service.bal -o social_media_service.yaml
